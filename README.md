## ⚡ Features

| Feature | Description |
|---|---|
| 🎨 **Dark Glassmorphism UI** | Deep space dark theme with glowing purple & pink accents |
| ✨ **Ripple on Click** | Every button press fires a radial ripple from your tap point |
| 💥 **Result Pop Animation** | Numbers spring to life with a satisfying spring bounce |
| 🌟 **Equals Flash** | The entire shell pulses with glow when you hit `=` |
| ⚠️ **Error Shake** | Divide by zero? The display physically shakes |
| ⌨️ **Full Keyboard Support** | Type naturally — no mouse needed |
| 📱 **Responsive** | Works perfectly on mobile and desktop |
| 🔤 **Custom Fonts** | Syne + JetBrains Mono for that designer edge |

---

## ⌨️ Keyboard Shortcuts

```
0–9        →  Input numbers
+ - * /    →  Operators
Enter      →  Calculate
Escape     →  Clear all
Backspace  →  Delete last digit
.          →  Decimal point
%          →  Percentage
```
